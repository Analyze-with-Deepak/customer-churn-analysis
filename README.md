
# Customer Churn Analysis

## Overview

This project analyzes customer churn data to identify key factors that influence customer retention and provide business-focused recommendations to reduce churn.

## Tools Used

* Python (Pandas, Matplotlib)
* R (statistical testing, logistic regression, decision tree)
* Google Colab & Posit Cloud

## Key Findings

* Customers with **shorter tenure** are more likely to churn
* **Higher monthly charges** are associated with increased churn
* **Month-to-month contracts** show higher churn risk compared to long-term contracts

## Approach

* Cleaned and explored data using Python
* Performed statistical tests (t-tests) in R
* Built models (logistic regression and decision tree) to analyze churn drivers
* Created segment-level analysis to identify high-risk customer groups

## Business Recommendations

* Target high-risk customers (e.g., month-to-month users) with retention offers
* Improve onboarding experience for new customers
* Review pricing strategy for high-charge customer segments

## Project Outcome

This analysis highlights key drivers of churn and demonstrates how data can be used to support customer retention strategies.
